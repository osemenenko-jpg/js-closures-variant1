## Опис

У проєкті реалізовано приклади використання замикань у JavaScript.

Замикання використовуються для створення приватного стану, який недоступний напряму ззовні.

## Реалізовано

- `createCounter(initial)`
- `createLimitedCounter(min, max)`
- `createStepCounter(step)`
- `memoize(fn)`
- `memoizeExpiring(fn, ttlMs)`
- benchmark для Fibonacci

## Використані патерни

### Counter Factory

Лічильники створюються через функції-фабрики.  
Змінна `count` є приватною і доступна тільки через методи.

### Мемоізація

Результати функцій зберігаються у приватному кеші.  
При повторному виклику з тими самими аргументами результат береться з кешу.
